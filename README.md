# 🌍 Collafric | Carry a Piece of Culture

**Collafric** es una landing page de e-commerce premium dedicada a la joyería artesanal de inspiración africana. Este proyecto combina un diseño visual potente con una experiencia de usuario fluida y moderna.

---

## ✨ Características principales

* **Video Hero Inmersivo:** Cabecera a pantalla completa con video en loop para captar la atención desde el primer segundo.
* **Glassmorphism Navigation:** Barra de navegación moderna con efecto de desenfoque XL y bordes sutiles, inspirada en las últimas tendencias de UI.
* **Carrito de Compras Funcional:** Gestión de estado en JavaScript (Vanilla) que permite añadir/quitar productos y visualizar el total en un menú lateral (Offcanvas).
* **Diseño Masonry:** Galería de comunidad estilo Instagram que adapta las imágenes de forma dinámica para un look de mosaico profesional.
* **Responsive Design:** Totalmente adaptado a dispositivos móviles y optimizado para pantallas de escritorio utilizando el sistema de rejilla de Bootstrap 5.

## 🚀 Tecnologías utilizadas

* **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
* **Framework CSS:** [Bootstrap 5.3](https://getbootstrap.com/) para la estructura y componentes interactivos.
* **Diseño:** [Figma](https://www.figma.com/) para el prototipado inicial y la arquitectura visual.
* **Iconos:** [Bootstrap Icons](https://icons.getbootstrap.com/).
